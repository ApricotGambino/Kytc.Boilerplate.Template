# Guides Overview

The *how do I write this code?* part of the documentation.

>If you understand what a [Type Table](../articles/type-table.md) entity is, but want to know how to add one, then check [How To Add A Type In The Type Tables](how-to-add-a-type-in-the-type-tables.md).  These articles will link to other relevant articles.
